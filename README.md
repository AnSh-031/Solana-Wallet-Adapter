# Solana Wallet Adapter DApp (React)

This is a simple Solana DApp built using **React** and the **Solana Wallet Adapter**.  
It allows users to connect their Solana wallet, view their SOL balance, request a Devnet airdrop, send SOL to another wallet, and sign messages.

---

## Features

- Connect and disconnect Solana wallets (Phantom, Solflare, etc.)
- Display the connected wallet's SOL balance
- Request free SOL airdrop on Devnet
- Send SOL tokens to any valid wallet address
- Sign and verify messages using the wallet

---

## Tech Used

- **React**
- **Solana Wallet Adapter**
- **Solana Web3.js**
- **Noble Ed25519**
- **bs58**
